Arrays

Arrays are special variables that can hold more than one value at a time. They are list-like objects. Within this special object, you can list multiple variables. This will be useful in other aspects coding later on! Below explains how to create an array out a list.


let fruit1 = "Apple";

let fruit2 = "Orange";

let fruit3 = "Banana";


Now let's make an array!


let fruits = ["Apple","Orange","Banana"];

It will be useful to use Arrays in certain coding circumstances!


Conditionals

For Loops

Loops are pretty nifty if you need to run the same code over and over again with different values. In this case, for Loops runs a block of code over and over again until it hits a statement at that is not true! Below is the syntax!


for (variable; condition; statement) {

// block of code to run }
